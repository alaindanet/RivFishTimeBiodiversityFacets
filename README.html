Installation

You can install the released version of RivFishTimeBiodiversityFacets from CRAN with:

install.packages("RivFishTimeBiodiversityFacets")

And the development version from GitHub with:

# install.packages("devtools")
devtools::install_github("alaindanet/RivFishTimeBiodiversityFacets")
